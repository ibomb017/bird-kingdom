# 鸟鸟王国 — 系统说明（开发环境与配置方式）

## 一、项目概述

"鸟鸟王国"是一款基于深度学习的鸟类语音识别与饲养管理iOS App，包含以下四个子系统：

| 子系统 | 技术栈 | 代码行数 | 说明 |
|--------|--------|---------|------|
| iOS客户端 | Swift + SwiftUI | ~57,700行 | 面向终端用户的移动应用 |
| 后端API服务 | Swift + Vapor 4 | ~13,000行 | RESTful API后端 |
| 管理端后台 | Java SpringBoot + HTML/JS/CSS | ~15,600行 | Web管理后台（前后端） |
| 短信代理服务 | Java SpringBoot | ~900行 | 阿里云短信验证码代理 |

**总代码量：约87,200行**

## 三、环境配置与运行步骤

### 3.1 数据库配置

```bash
# 1. 安装 MySQL 8.0
brew install mysql@8.0  # macOS
# 或
sudo apt install mysql-server  # Ubuntu

# 2. 创建数据库
mysql -u root -p
CREATE DATABASE bird_kingdom CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 3. 数据库迁移由Vapor自动执行（启动时自动创建表）
```

### 3.2 后端API服务启动

```bash
cd backend-swift

# 1. 复制环境变量配置文件
cp .env.example .env

# 2. 编辑 .env，填入数据库连接信息
#    DB_HOST=127.0.0.1
#    DB_PORT=3306
#    DB_USER=root
#    DB_PASSWORD=your_password
#    DB_NAME=bird_kingdom
#    JWT_SECRET=your_jwt_secret

# 3. 编译并运行
swift build
swift run App serve --hostname 0.0.0.0 --port 8080
```

### 3.3 iOS客户端运行

```bash
cd frontend/BirdKingdom

# 1. 用Xcode打开项目
open BirdKingdom.xcodeproj

# 2. 修改API地址配置（如需要）
#    文件: Services/AppConfig.swift
#    将 baseURL 改为本地后端地址：http://localhost:8080

# 3. 选择模拟器或真机，点击 Run (⌘R)
```

### 3.4 管理端后台运行

```bash
cd admin-web

# 后端
cd backend
mvn spring-boot:run

# 前端（新终端窗口）
cd admin-web
npm install
npm run dev
# 访问 http://localhost:5173
```

### 3.5 短信代理服务运行

```bash
cd sms-proxy

# 1. 复制配置文件
cp .env.example .env
# 2. 填入阿里云短信 AccessKeyId、AccessKeySecret

# 3. 运行
mvn spring-boot:run
```

## 四、线上部署环境

项目已部署至阿里云服务器，线上环境配置如下：

| 组件 | 部署方式 | 地址 |
|------|---------|------|
| Vapor后端 | Docker容器 | api.birdkingdom.xyz:8080 |
| 管理端后台 | Docker容器 | admin.birdkingdom.xyz:8082 |
| 短信代理 | Docker容器 | 内网 :8083 |
| MySQL数据库 | 阿里云RDS | 内网连接 |
| 图片存储 | 阿里云OSS | oss-cn-shanghai |
| Web服务器 | Nginx | 反向代理 + HTTPS |
