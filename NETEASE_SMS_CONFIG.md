## 🔍 如果需要自定义模板

### 在网易云信控制台创建模板：

1. **登录控制台**
   ```
   https://app.yunxin.163.com/
   ```

2. **找到短信模板**
   ```
   可能的位置：
   - 短信验证码 -> 模板管理
   - 产品服务 -> 短信验证码
   - 应用管理 -> 短信配置
   ```

3. **创建模板**
   ```
   模板类型：验证码
   模板内容：您的验证码是${code}，5分钟内有效，请勿泄露给他人。
   ```

4. **获取模板ID**
   ```
   审核通过后会得到模板ID（8位数字）
   ```

5. **更新配置**
   ```yaml
   netease:
     sms:
       template-id: 你的模板ID
   ```

## 🔧 故障排查

### 1. 短信发送失败

**检查AppKey和AppSecret：**
```
控制台 -> 应用管理 -> 应用信息
确认配置是否正确
```

**检查余额：**
```
控制台 -> 账户管理
确保有足够余额
```

**检查模板ID：**
```
确认模板ID是否正确
确认模板已审核通过
```

### 2. 收不到短信

**检查手机号格式：**
```
必须是11位中国大陆手机号
格式：13800138000
```

**检查发送频率：**
```
同一手机号60秒内只能发送一次
```

**检查运营商屏蔽：**
```
部分运营商可能屏蔽验证码短信
尝试换个手机号测试
```

### 3. CheckSum错误

**确认时间同步：**
```
服务器时间必须准确
检查系统时间是否正确
```

**确认AppSecret：**
```
AppSecret不能有多余空格
复制时注意完整性
```

## 🚀 双重保障机制

当前配置实现了双重保障：

```
1. 优先使用网易云短信
   ↓ 成功
   ✅ 发送成功

   ↓ 失败
2. 自动降级到阿里云短信
   ↓ 成功
   ✅ 发送成功

   ↓ 失败
3. 控制台打印验证码
   ✅ 开发环境可用
```

这样即使短信服务出问题，开发测试也不受影响！
